# Tire-Identifier
An Image Processing App build for android device using python kivy with Template Matching Method

requirement:

- Python v3.x
- Kivy (https://kivy.org/)
- Buildozer (https://buildozer.io/)

Python-Pip:

- Pillow (PIL)
- Numpy

build APKs:

`$ buildozer -v android debug`
